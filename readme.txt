Woocommerce Recently purchased items grid
=========================================

This simple plugin showcases the currently logged in users recent purchases in a standard woocommerce product grid.

The grid can be configured via Settings -> Purchase History Grid

The plugin uses a standard wordpress shortcode to display the grid so that it can be embedded using any wordpress page builder.
